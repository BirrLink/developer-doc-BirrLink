# BirrLink Developer Documentation

Welcome to the BirrLink developer documentation. This documentation provides comprehensive guides, API references, and resources to help you integrate with the BirrLink payment gateway.

## About BirrLink

BirrLink is Ethiopia's premier payment gateway, enabling businesses to accept payments securely and efficiently. Our platform supports multiple payment methods including mobile money, bank cards, and other local payment options.

## Documentation Structure

The documentation is organized into the following sections:

- **Getting Started**: Introduction to BirrLink and quickstart guides
- **API Reference**: Complete API documentation with request/response examples  
- **Payment Methods**: Detailed guides for each supported payment method
- **Webhook Events**: Information on handling payment events and notifications
- **SDKs & Libraries**: Language-specific libraries for popular programming languages
- **Testing & Sandbox**: Guidelines for testing your integration
- **Security**: Best practices and compliance information

## Local Development

To run this documentation site locally:

1. Ensure you have Node.js (version 20 or higher) installed
2. Navigate to the documentation directory
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open your browser to `http://localhost:3000`
